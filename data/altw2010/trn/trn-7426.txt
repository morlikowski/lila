'''『サルゲッチュ3』'''は、2005年7月14日にソニー・コンピュータエンタテイメントから発売されたプレイステーション2専用ソフトである。
==登場キャラクター==
''サルゲッチュの登場キャラクターを参照''
==へんしん==
''サルゲッチュ#へんしんを参照''
==前作との違い==
*拠点となる「'''トラベルステーション'''」の名称が「'''テレビステーション'''」に変更されている。
*「ガチャボックス」は廃止されたが、「商店街」で買い物をできる。
*「トレーニングルーム」に「へんしん」を追加。
===ガチャメカ===
*前作よりガチャメカの数が大幅に減った。
*1、2に登場した「マジックパンチ」が登場しない。
==クリア特典==
*1周目をクリアするとタイトルに「サばいばル」が出現する。

'''捉猴啦3'''，於2005年7月14日由新力電腦娛樂發行之PS2遊戲。
==概要==
與曾是博士的學生，也是科學家秋惠伯母每天過著和平日子，活力充沛的小男孩，小悟，與他的姐姐，當紅偶像的彩香。有一天，嗶波猴忽然佔據了世界各地的電視台，並開始播放一些「超猴的節目」!並由於內容過度「無聊」，會令持續觀看的人都變成懶惰蟲!且這件事背後，又有戴上嗶波帽的斯貝達存在!甚且，他似乎還與誓言對人類復仇的神秘協力者，智腦巫奇博士聯手合作。再不快點阻止那些奇怪的電視節目的播放，世界就要被嗶波猴支配了!

代替因嗶波猴的節目而變成懶惰蟲的博士，小翔，小光，這次改由小悟和彩香挑戰捉猴啦的大冒險!
==登埸人物==
;  小悟(サトル)
:  (聲優:竹內順子)
:  本遊戲的主角，最喜歡體育與午餐時間的男孩。熱愛棒球，且總是穿著喜愛的制服。
;  彩香(サヤカ)
:  (聲優:野中藍)
:  本遊戲的另一主角。小悟的姐姐，且是正紅遍半邊天的偶像。擅長唱歌，運動神經也出類拔群。
;  秋惠(アキエ)
:  (聲優:大原さやか)
:  曾擔任博士助手的科學家，與小悟和彩香一起生活。
;  斯貝達(スペクター)
:  (聲優:坂本千夏)
:  嗶波猴的領導，被小光捉住後，再度戴上嗶波帽，且企圖再支配世界。
;  智腦巫奇博士(Dr.トモウキ)
:  (聲優:藤原啓治)
:  對人類心懷憎惡的科學家，協助斯貝達，誓言對人類復仇。
:  IQ1300，喜歡的字彙是「完美」。

==遊戲內容==
遊戲設有20個關卡及7個打倒巫奇的關卡，在普通關卡裡如果設有偷窺攝影機的話，可以拍攝猴子的電影。
