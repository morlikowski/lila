== Including the POTD on your user page ==
You can insert the image on your user page or talk page by adding the text <code></code> (this version is shown above). It will be displayed in a box of no more than 600 pixels wide, and will include a small amount of text based on the article in which the image appears. A no-text version is also available via <code></code>.

It is also possible to create your own custom POTD layouts, in case the already-existing versions will not look good within your user page design. Mix and match the following components to make your own. '''This system will only work for Pictures of the day selected beginning January 1, 2007.''' Be sure to replace <code>[date]</code> with an appropriate date value. For a dynamically updating version, use <code><nowiki></nowiki></code> (example: <code><nowiki>|image}}</nowiki></code>).

There are two additional pre-defined layouts
*<code><nowiki>|column}}</nowiki></code>: This has the image and caption above each other with no borders. Used on some of the Main Page alternatives and suitable for user pages as well.
*<code><nowiki>|row}}</nowiki></code>: This has the image and caption  encapsulated in (usually) a single table row. This is the version used on the Main Page.

It is also possible to permanently feature a POTD for a selected day. Just add a specific value for the date you want. For example, today's POTD is <code>--|image}}</code>. Likewise, you can use date parameters with the other templates as well. If you like the pre-made formats, you can use date parameters there as well, like this: <code><nowiki>--}}</code> or <code><nowiki>--}}</code>. Again, '''this system will only work for Pictures of the day selected beginning January 1, 2007.'''
