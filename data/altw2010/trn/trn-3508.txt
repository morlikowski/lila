'''Microsoft Excel''' (engl. []) este un program de calcul tabelar. Astazi este cel mai utilizat Software pentru calcul tabelar.
Excel aparţine grupei de programe Microsoft Office şi este disponibilă pentru Microsoft Windows cât şi pentru Mac OS. Actuala versiune disponibilă este pentru ''Microsoft Excel 2007'' (din 30 noiembrie 2006 pentru firme respectiv  30 ianuarie 2007 für utilizatori) ca şi pentru Mac OS ''Microsoft Excel 2008'' (din ianuarie 2008).

== Istoric ==

* 1982 Microsoft Multiplan
* 1985 prima versiune pentru Apple Macintosh urmare la Microsoft Multiplan
* 31 octombrie 1987 Excel&nbsp;2.0 – prima versiune pentru IBM-kompatible PCs (sistemul de operare DOS 3.0 sau Microsoft Windows 2.0) 
* 12 septembrie 1990 Excel 3.0 (sistemul de operare Microsoft Windows 3.x)
* 1 aprilie 1992 Excel 4.0
* 14 decembrie 1993 Excel 5.0
* 14 decembrie 1994 Excel 5.0 pentru Windows NT – prima versiune 32-Bit
* 27 iulie 1995 Excel 95 (versiunea 7.0)
* 1997 Excel 97 (versiunea 8.0)
* 1998 Excel 98 pentru Macintosh
* 1999 Excel 2000 (versiunea 9.0)
* 2001 Excel:mac 2001 (ultima versiune pentru Mac OS 8 şi 9)
* 2001 Excel 2002 (versiunea 10.0, Office-Suite numită Office XP)
* 2001 Excel:mac v. X (prima versiune pentru Mac OS X)
* 21 octombrie 2003 Office Excel 2003 (versiunea 11.0)
* iunie 2004 Excel:mac 2004 – ultima versiune pentru PowerPC Macs
* noiembrie 2006 Office Excel 2007 (versiunea 12)
* ianuarie 2008 Excel:mac 2008 – versiune actuală pentru PowerPC- şi Intel-Macs

'''Microsoft Excel''' (полное название '''Microsoft Office Excel''') — программа для работы с электронными таблицами, созданная корпорацией Microsoft для Microsoft Windows, Windows NT и Mac OS. Входит в состав пакета Microsoft Office.

== Версии ==

* 2001 — Excel 2002 (10) — Microsoft Office XP

== История создания ==
Microsoft изначально продавала табличный процессор, названный GUI software. Microsoft pushed its advantage with regular new releases, every two years or so. Текущая версия для Windows  - Excel 12, также называется Microsoft Office Excel 2007. Текущая версия для Mac OS X — Microsoft Excel 2004. -->
